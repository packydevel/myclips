MyClips
==============

A Rete-based, CLIPS clone, inference engine in Python.


Dependencies:
------------

	BList (>=1.3.4):
		https://github.com/DanielStutzbach/blist or `easy_install blist`
	
	NetworkX and MathplotLib (for Rete network plotting only)
